# 🇵🇭 Philippine Language Translator

A lightweight open-source web translator supporting multiple Philippine languages.

## 🌏 Supported Languages
- Filipino / Tagalog
- Kapampangan
- Ilocano
- Cebuano
- Waray

## 🎯 Purpose
This project aims to make Philippine regional languages more accessible and help preserve linguistic diversity through a simple browser-based translator.

## 🚀 Live Demo
https://toolsphere.github.io/translator/

## ⚙️ Features
- Instant translation (no translate button needed)
- Language dropdown selection
- Mobile and desktop friendly
- Copy and clear buttons

## 🛠 Built With
- HTML
- CSS
- JavaScript
